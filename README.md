# TableCRM - Форма создания карточки товара

Приложение для создания карточек товаров в системе tablecrm.com, разработанное с использованием Next.js, TypeScript и shadcn/ui.

## Технологии

- **Next.js 16** - React фреймворк с SSR
- **TypeScript** - типизация
- **Tailwind CSS v4** - стилизация
- **shadcn/ui** - UI компоненты
- **React Hook Form** - управление формами
- **Zod** - валидация форм

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

## Сборка

```bash
npm run build
```

## Деплой на Vercel

Проект готов к деплою на Vercel:

1. Подключите репозиторий к Vercel
2. Vercel автоматически определит настройки Next.js
3. Деплой произойдет автоматически

Или используйте Vercel CLI:

```bash
npm i -g vercel
vercel
```

## Использование

1. Откройте форму создания карточки товара
2. Заполните обязательные поля (название товара)
3. Заполните дополнительные поля по необходимости
4. Нажмите "Создать карточку товара"
5. Токен для API можно передать через URL параметр `?token=YOUR_TOKEN`

## API

Форма отправляет данные на `/api/create-product`, который проксирует запрос к API tablecrm.com.
